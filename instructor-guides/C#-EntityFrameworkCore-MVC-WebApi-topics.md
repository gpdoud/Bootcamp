# C#, EntityFrameworkCore, MVC, WebApi

## C#

* statements
    * statements end with semi-colons
    * statement blocks with {}
    * comments
        * // comment to end of line
        * /* .. */ multiline comments
        * /// special docs
    * case sensitive
* basic program 
    * using
    * namespace
    * class : contains all code
    * Main method
* variables
    * type name { = value};
        * class variables are initialized to default values
        * local variable must be initialized
    * var : type inference
    * scope : declared inside braces exists till closing brace
* const int x = 1; // can never change; they are static
* data types
    * value types: 
        * integers: int, long
        * floating point: float, double, decimal
        * boolean: bool
        * character: char
        * string: string
            * @ : can us single back-slash
            * $ : interpolation
    * reference types
* program flow
    * equality test : ==
    * if .. else
    * switch .. case .. default
    * loops
        * for(;;)
        * while()
        * do .. while()
        * foreach()
        * break
        * continue
        * return
* namespaces
    * alias : using alias = Namespace
* Main method
    * must exist in one class
    * must be static
    * may return void or int
    * may have string[] parameter
